##  Recsys-online

##  What is it? <br />  
  Recsys-online 是一个推荐系统的在线计算部分，接收，存储日志并生成用户偏好，它： <br /> 

  1.基于Kafka，Storm，HBase，Zookeeper开发的 <br /> 
  2.目前只支持的流程：从Kafka中读取数据==>Storm数据处理===>HBase存储<br /> 
  -----------

  The Current Version : 1.0-beta <br />

  -----------

##  Package : <br />  
  mvn clean package
  
  -----------

##  Run : <br />  
  cd target<br />  
  tar xvf recsys-online-bin.tar.gz<br />
